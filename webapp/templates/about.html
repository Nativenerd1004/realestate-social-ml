Pipeline Summary

  • 13 Kaggle datasets fetched automatically via kagglehub API
  • 45,533 LA Airbnb listing records — primary training dataset
  • 9 classifiers trained with 80/20 stratified split
  • 10 auto-retrain iterations with GridSearchCV
  • Best model: Random Forest — Accuracy 84.1%, ROC-AUC 0.85
  • MinMaxScaler feature normalization across 23 features
