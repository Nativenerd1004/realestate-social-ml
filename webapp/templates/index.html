STAR Method

S
Situation

Real estate platforms face host churn, poor listing pricing strategy, uncertain sale outcomes, and ineffective social media content — all costly problems with no data-driven tools to address them simultaneously.

T
Task

Build a multi-model ML pipeline that ingests real Kaggle datasets, engineers features, trains 8 classifiers per prediction target, and deploys a 4-tab interactive prediction app on Render.com.

A
Action

Fetched 5 real datasets via Kaggle API, applied SMOTE for class imbalance, trained RF/XGB/GBM/LR/DT/KNN/NB/AdaBoost per tab, selected best by F1, serialized with joblib, and deployed via GitHub → Render auto-deploy.

R
Result

Best overall: XGBoost on YouTube data — F1=0.8762, ROC-AUC=0.9465. Full 4-tab prediction suite live at realestate-social-ml.onrender.com covering 1.9M+ real training records across USA and Canada.
