STAR Method

S
Situation

Real estate platforms face high host churn, where property owners become inactive and stop listing — reducing inventory and platform revenue. Social media engagement patterns can signal early churn indicators.

T
Task

Build an autonomous ML pipeline that ingests real-world datasets from Kaggle, cleans and processes the data with PostgreSQL, trains 9 classification models, and auto-retrains until F1 and ROC-AUC targets are reached.

A
Action

Developed a 10-step Python agent: Kaggle fetch → PostgreSQL load → data cleaning → feature engineering → model training (RF, XGB, GBM, LR, DT, SVC, KNN, NB, AdaBoost) → evaluation → auto-retrain loop → dashboard → deploy.

R
Result

Best model: Random Forest with 84% accuracy, 87% precision, ROC-AUC of 0.85. Fully deployed to GitHub and Cloudflare Pages. Interactive prediction module allows real-time host churn scoring.
