Why Random Forest Won

Random Forest achieved the highest precision (87.3%) and ROC-AUC (0.85), making it the most reliable model for identifying true churn cases while minimising false positives — critical for a business use case where wrongly flagging an active host is costly.

F1 Score Context

F1 scores in the 0.59 range reflect significant class imbalance — churned hosts are a minority. The next iteration will apply SMOTE oversampling and a more focused churn dataset to push F1 towards the 0.90 target.

Hyperparameter Tuning

Each of the 10 retrain iterations applied GridSearchCV with 3-fold StratifiedKFold cross-validation. Progressive parameter grids were used — broader search in early iterations, finer tuning in later ones.
