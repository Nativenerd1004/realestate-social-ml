SMOTE for Class Imbalance

The Host Churn target had only 2.85% positive class — severely skewed. Synthetic Minority Oversampling (SMOTE) was applied before training to balance classes, improving F1 from 0.59 to 0.77.

4 Independent Models

Each tab uses a separately trained model, scaler, and feature set. Host Churn (285k CRM rows), Market Position (235k US+Canada listings), Listing Sale (8,574 Illinois sales), Social Media (178k YouTube trending videos).

Why XGBoost Dominates

XGBoost wins 3 of 4 tabs — gradient boosting handles mixed feature types, missing values natively, and finds non-linear patterns that single trees and linear models miss. The Social Media tab hits F1=0.88 and ROC=0.95.
